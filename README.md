# Project Description
The User Authentication Module is a secure system designed to manage
user registration and login functionality. It ensures that only
authorized users can access the system by validating credentials
securely.

## Features
- User Registration
- Secure Login System
- Password Hashing with Salt
- Security Question Verification
- Login Attempt Limit
- File-based Data Storage

## Technologies Used
- C Programming Language
- File Handling
- Data Structures
- Hashing Techniques

## Objective
The main objective of this project is to understand how authentication
systems work in real-world applications and how user data can be
secured.

## How to Run
1. Open the source code in a C compiler
2. Compile and run the program
3. Use the menu options to register and log in
